MorphHomeX is a neumorphic theme for Home Assistant featuring soft shadows, rounded surfaces, and a tactile 3D UI feel.

It includes 12 variants across two styles:

- **Raised** (default neumorphic look): light, dark, dark-taupe, dark-graphite, dark-ash, dark-onyx
- **Inset** (recessed elements): light-inset, dark-inset, dark-taupe-inset, dark-graphite-inset, dark-ash-inset, dark-onyx-inset

Requires [card-mod](https://github.com/thomasloven/lovelace-card-mod) installed via HACS.
